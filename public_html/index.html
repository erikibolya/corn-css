Corn CSS

Conventions

Corn CSS has no strict uses.
Function rem() helps with calculations as follows. 
.class{ font-size: rem(20); }
generates 
.class{ font-size: 1.25rem; }

Typography

Typography across all the framework is defined by rem and em units and is completely fluid. Font size of body is set to 100% so 1 rem is 16 px (browser sets default font size as 16px, if user manipulates this value and sets it to 18px, 1rem will equals 18px).
Function rem() helps with calculations as follows. 
.class{ font-size: rem(20); }
generates 
.class{ font-size: 1.25rem; }
